# Sex Differences in Diabetes Risk Factors: NHANES Analysis

## 🔍 Key Finding

**Women in perimenopausal years (45-55) show a 4.4-fold increase in diabetes prevalence** compared to pre-menopausal women (p < 2.2e-16)

## 📊 Files

- **nhanes_diabetes_risk_analysis.Rmd** - Complete analysis code
- **nhanes_diabetes_risk_analysis.html** - Download and open in browser to see visualizations

## 🧬 About

Sex-stratified epidemiological analysis of NHANES data examining diabetes risk factors across reproductive life stages in women.

## 📈 Methods

- Data: NHANES (n=4,178 after cleaning)
- Analysis: t-tests, chi-square, stratified by gender and reproductive stage
- Tools: R, tidyverse, ggplot2, gtsummary
